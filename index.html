HTML

  • The head element contains information about the webpage
  • the body element represents the visible content shown to the user
  • src: describes loaction/source of image
  • alt: describes image and its functionality for accessibility purposes (screen readers)
  • child elements: elements nested within other elements (i.e. anything within body is chilid to body, lists child to unordered lists, etc.)
  • parent elements: elements with other elements nested within them (i.e. body is parent element)
  • declarations are NOT case-sensitive (i.e. doctype); elements ARE case-sensitive

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The box-shadow CSS property adds shadow effects around an element's frame.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • cd: change directory
  • cd ..: change directory backwards
  • ls: list files and directories
  • pwd: print working directory
  • mkdir: make a new directory
  • touch: create new file
  • code .: start VS within chosen directory/branch
  • git revert [SHA]: reverts to previous version
  • git clone: creates local copy
  • git remote -v: checks connection with repository

JavaScript

  • var: creates variables
  • strict equality operator (===) checks to see if a value is true or false (boolean)
  • console.log() adds a message to the web console
